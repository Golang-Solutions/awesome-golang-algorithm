# [709. To Lower Case][title]

## Description

Implement function ToLowerCase() that has a string parameter str, and returns the same string in lowercase.

**Example 1:**

```
Input: "Hello"
Output: "hello"
```


**Example 2:**

```
Input: "LOVELY"
Output: "lovely"
```

**Example 3:**

```
Input: "here"
Output: "here"
```
