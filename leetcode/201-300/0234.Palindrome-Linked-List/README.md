# [234.Palindrome Linked List][title]

## Description
Given the `head` of a singly linked list, return true if it is a palindrome.

**Example 1:**  
![exampl1](pal1linked-list.jpg)

```
Input: head = [1,2,2,1]
Output: true
```

**Example 2:**  
![example2](pal2linked-list.jpg)

```
Input: head = [1,2]
Output: false
```
